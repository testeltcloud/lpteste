# Imagens Necessárias

Esta pasta deve conter as seguintes imagens para o projeto funcionar corretamente:

1. **logo-todagente.png**
   - Logo da Todagente Telemedicina
   - Tamanho recomendado: 200x60px
   - Fundo transparente

2. **logo-norte2020.png**
   - Logo do Programa Norte 2020
   - Tamanho recomendado: 150x50px
   - Fundo transparente

3. **logo-portugal2030.png**
   - Logo Portugal 2030
   - Tamanho recomendado: 150x50px
   - Fundo transparente

4. **logo-uniao-europeia.png**
   - Logo da União Europeia com texto "Cofinanciado pela União Europeia"
   - Tamanho recomendado: 200x60px
   - Fundo transparente

## Imagens de Conteúdo

5. **doctor.png**
   - Imagem da médica com estetoscópio
   - Tamanho recomendado: 600x700px
   - Fundo transparente (PNG)

## Notas

- Todas as imagens devem estar em formato PNG para melhor qualidade
- Use compressão de imagem para otimizar o carregamento
- Mantenha a proporção adequada para evitar distorções
